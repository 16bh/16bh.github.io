# css取色器 - ColorPicker

将光标置于想要输入颜色的位置，打开`sublime`的控制台，输入`color picker`召唤出取色器，选择你想要的颜色，点击`pick`（也可以设置快捷键）

![](sublime-plugin-css-color-hignlighter/1240-20181016235907106.png)
